## CSS Text Transformation

---

### 텍스트 변환

text-transform속성은 대문자와 텍스트에서 소문자를 지정하는 데 사용됩니다.

모든 것을 대문자 또는 소문자로 바꾸거나 각 단어의 첫 글자를 대문자로 바꾸는 데 사용할 수 있습니다.

    예시
    p.uppercase {
    text-transform: uppercase;
    }

    p.lowercase {
    text-transform: lowercase;
    }

    p.capitalize {
    text-transform: capitalize;
    }
