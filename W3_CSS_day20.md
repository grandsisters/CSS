## CSS 둥근 테두리

***
### CSS 둥근 테두리
border-radius속성은 요소에 둥근 테두리를 추가하는 데 사용됩니다

<img src='./img/css_borders.png'>

예시
p {
  border: 2px solid red;
  border-radius: 5px;
}
